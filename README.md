# Vehicle Booking System

This project is a simple vehicle booking system built using React and Material-UI (MUI). The system allows users to select a vehicle type, then a specific vehicle model, and finally fill out a booking form to book the vehicle.

## Features

- Select vehicle type (Car or Bike)
- Select vehicle model based on the chosen type
- Fill out the booking form with personal details and booking dates
- Responsive and modern UI using Material-UI components
